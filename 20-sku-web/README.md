# SKU 重量核验 Web 工具

直接打开 `index.html` 即可使用。页面不依赖外部 CDN，可离线导入 CSV / XLSX。

## 表格格式

预估重量表至少包含两列：

```csv
SKU,预估重量(g)
SKU-001,125
SKU-002,248.5
```

实际称重批量导入至少包含两列：

```csv
SKU,实际重量(g)
SKU-001,151
SKU-002,250
```

工具会自动识别常见中文和英文表头。默认阈值为 `20g`，误差大于等于阈值会弹窗警示，并进入异常 SKU 清单。

## 离线部署

部署时上传这些文件到同一个目录：

- `index.html`
- `styles.css`
- `app.js`

说明：离线 XLSX 解析依赖新版浏览器内置能力，建议使用 Chrome / Edge。旧版 `.xls` 请先另存为 `.xlsx` 或 `.csv`。
